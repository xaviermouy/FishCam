2- install software:

sudo apt upgrade

3- Dowload GitHub repo:
cd /home/fishcam/Desktop/
git clone https://github.com/xaviermouy/FishCam.git.

4- Open control center:
Interfaces > enable ssh
Localization > set timezone to UTC
System > Boot to CLI
System > Change hostname to fishcam01

5- Setup cron job:
crontab -e
@ reboot sh/home/fishcam/Desktop/FishCam/script/camStartup.sh &
Save the changes by pressing CTRL  + O, then press Enter to confirm.
Exit Nano by pressing CTRL  + X
Verify that the schedule has been saved: crontab -l
